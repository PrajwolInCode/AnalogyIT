Starting a business: tech checklist

  • Domain name + professional email (Microsoft 365 / Google Workspace)
  • User accounts & staff access (don’t share passwords)
  • Backup plan (files + email) and recovery options
  • Wi‑Fi stability and device basics (printers, scanners, POS)
  • Website essentials: mobile friendly, clear services, contact form
  • Security basics: MFA, updates, safe password habits

Website basics that help SEO

  • One clear H1 per page + readable headings
  • Fast loading images (compressed)
  • Unique page titles and meta descriptions
  • Sitemap + Google Search Console submission
  • Google Business Profile for local searches

Microsoft 365 quick wins

  • Enable MFA for every account
  • Use shared mailboxes for team inboxes (sales@ / info@)
  • Set up recovery options and admin roles properly
  • Keep devices updated and protected

When Wi‑Fi is unreliable

  • Place the router centrally (not hidden in a cupboard)
  • Use a mesh system if coverage is weak
  • Separate guest Wi‑Fi from staff devices
  • Check cabling and interference sources
